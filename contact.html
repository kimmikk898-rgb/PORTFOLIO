Frequently Asked Questions

How long does a project take?
Simple landing pages take 3–7 days. Portfolio websites take 1–2 weeks. Management systems range from 2–6 weeks depending on complexity.
Do you offer payment in installments?
Yes. A 50% downpayment is required to start, and the remaining 50% is paid upon project completion before source code is released.
What if I need changes after delivery?
Every package includes 2 free revisions. Additional revisions beyond that are charged at ₱300–₱800 per revision depending on the scope.
Will I get the full source code?
Yes — full source code is provided after complete payment. You own the code and can modify it however you like after that.
Can you host the website for me?
I can guide you on deployment (Vercel, cPanel, etc.) but I don't manage hosting directly. I'll help you set it up step by step.
What tech stack do you use?
TypeScript, Tailwind CSS, Supabase, MySQL, Python, and C++. I recommend the best fit based on your project needs.
